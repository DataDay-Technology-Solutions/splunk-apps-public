# Splunk Innovators Toolkit

Production-ready UI components for building interactive Splunk dashboards and apps.

## What's Included

- **6 web components**: `sit-button`, `sit-checkbox`, `sit-modal`, `sit-table`, `sit-toast`, `sit-toggle`
- **7 walkthrough views**: Getting Started, Quick Start, Best Practices, Code Examples, Component Reference, Demo, Hands-On Exercises
- **Core styles**: production-tested CSS (core + components)
- **No external dependencies** — drop-in install, works with Simple XML

## Installation

See the [repo root README](../README.md) for install instructions.

## Version

**1.0.0** — initial public release (build 15)

## Compatibility

- Splunk Enterprise 8.x and 9.x

## Usage

After install, navigate to **Apps → Splunk Innovators Toolkit** in Splunk Web. Start with the *Getting Started* view for a guided tour, or jump to *Component Reference* for live examples of each component.
